# San Pablo National High School Website

Welcome to the official website repository for **San Pablo National High School (SPNHS)**. This is a modern, responsive website built with HTML, CSS, and JavaScript.

## 🏫 About

This website serves as the digital home for San Pablo National High School, providing information about:
- School mission, vision, and values
- Academic programs offered
- Faculty and staff
- Admissions information
- Contact details and location

## 🚀 Getting Started

### Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shierjanmarco-cloud/SPNHS.git
   cd SPNHS
   ```

2. **Open in browser:**
   - Simply open `index.html` in your web browser, or
   - Use a local server:
     ```bash
     python -m http.server 8000
     # or
     npx http-server
     ```

3. **View the website:**
   - Visit `http://localhost:8000` in your browser

### GitHub Pages Deployment

1. Go to your repository settings
2. Navigate to **Pages** section
3. Set source to **main branch**
4. Your site will be available at `https://shierjanmarco-cloud.github.io/SPNHS/`

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #003366;        /* Change school primary color */
    --secondary-color: #0066cc;      /* Change school secondary color */
    --accent-color: #ff9900;         /* Change highlight color */
    --text-color: #333;
    --light-bg: #f5f5f5;
}
```

### Content
- Update school information in `index.html`
- Modify text, images, and contact details
- Add faculty members with appropriate avatars

### Contact Information
Update the following in the Contact section:
- Phone number
- Email address
- Physical address
- School hours
